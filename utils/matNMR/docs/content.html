Version 3.9.94 (october 27 2009)


MatNMR is a highly flexible toolbox for processing 1D and 2D NMR and EPR spectra under MATLAB, creating high-quality 1D, 2D or 3D plots from the spectra and printing them in every type of format that is supported by MATLAB.

Proper reference should be given in papers and/or presentations whenever matNMR has
been used for processing data. Please use the following reference:

J.D. van Beek, J. Magn. Res. 187 (2007), 19-26

Platforms:
In principle any platform on which MATLAB is running.
The code has been tested thoroughly under Solaris and MS Windows 95.
On a Mac (MacOS 7.5 and 8.0) and on Windows NT it seems to work properly too.

Compilers (required products):
None, as nothing of the code has been compiled (no big improvements can be expected by doing so I feel as
most of the routines that are used are built-in routines and graphical routines).

Free downloadable sources:
All m-files can be downloaded either as a gzipped tar-file (UNIX) or as a normal zip file (PC, MAC).
Current size (version 3.8.43 including manual pages) is approximately 8 Mb.
