The source code of matNMR is available in two formats: either as a gzipped tar-file or as a normal zip file. On all files in the normal zip file a 'Unix2Dos' has been performed making sure all end-of-line characters are suited for working under MS Windows.






NOTE: 11-07-'05: The colours in matNMR are screwed up for Matlab 7 under Windows XP when using the "Windows XP" display
setting the display properties of UIcontrols are changed. One should either work in "Windows classic" display
mode or use the workaround provided by Mathworks.:
Make a file called "java.opts" in the $MATLAB/bin/win32 directory, where $MATLAB is e.g. c:\program files\MATLAB704.
In the file java.opts you only write "-Dswing.noxp=true". This should fix the problem.
