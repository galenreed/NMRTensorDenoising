Finally there is a manual for matNMR! I hope you like it and that it is of any use. If not, please let me know. Besides reading this manual over the web you can read it from your local disk: All matNMR HTML pages are included with the code starting matNMR version 2.6. Just click on the "Help desk" entry in the menubar and your browser will be opened with the manual appearing.



For novel users there is a short tutorial with a 1D and a 2D dataset to quickly see how to process with matNMR. The tutorial can be found in paragraph 4.1



A PDF version of the manual is available here: matNMR.3.9.0.pdf. Sorry for the poor quality but it's difficult to convert HTML to PDF. You're very welcome to improve the manual!!!
